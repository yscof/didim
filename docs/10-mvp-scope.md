# 10. MVP Scope

## 선별 기준

새로 제안된 기능과 마케팅 아이디어는 아래 기준에 직접 기여하는 경우에만 MVP에 포함한다.

- 챌린지 완료율을 높이는가
- 다음 주 재방문율을 높이는가
- 사용 편의성과 금융 초보자의 이해 부담을 낮추는가
- 9월 MVP 기간 안에 구현 가능한가

## MVP에 적용

### 앱 기능

- 간편 진단 후 첫 챌린지 바로 제안
- 주 1개 챌린지 기본값
- Engage, Investigate, Act, Reflect 진행 화면
- 단계별 체크리스트형 실행 안내
- 외부 금융 서비스 링크는 제공하되 가입 대행은 하지 않음
- 완료 상태 3분류: 계획 완료, 실행 완료, 보류
- 보류 사유 기록과 다음 시도 예약
- 완료 후 효과 계산: 월 절감액, 연간 절감액, 저축 목표, 예상 달성 기간
- 스트릭(주간 기준), 누적 완료 수, 성장 기록
- 도구: 가계부 (자가 입력, 로컬 저장 — 디지털 월세·카드 재테크는 후속 차수)
- 금융 여정 지도: 출발지 + 3개 지역, 완료 시 오브젝트 변화, 지역 잠금·해금
- 배지 10개(행동 5, 지역 3, 꾸준함 2)와 성장 기록형 배지 화면
- 완료 리액션 5단계: 완료 인정, 금융 효과, 지도 변화, 배지, 다음 챌린지
- 지킨 돈/예약된 돈 이중 카운터
- 지난 방문 이후 변화 카드
- 주간 금융 행동 요약
- 회고 질문과 다음 챌린지 후보 제안
- AI의 개인화 칭찬과 다음 행동 1개 제안
- 추천 근거 표시: 사용된 입력, 매칭된 룰, 이번 주 추천 이유
- 상황 기반 CRM: 온보딩 미완료, 챌린지 미완료, 보류 후 재시도, 월급일, 주간 요약

### 초기 상세 구현 챌린지

- 비상금 목표 만들기
- 월급날 자동이체 계획 만들기
- 고정지출 한 개 줄이기
- 청약통장 상태 점검하기
- 현실적인 월 저축률 정하기
- 청년 자산 형성 정책 자격 간이 확인하기

### MVP 코어 후보 풀

별도 챌린지 라이브러리 v1에는 온보딩 2개와 MVP 코어 16개가 정리되어 있다. 첫 배포에는 위 6개를 상세 구현 슬라이스로 두고, MVP 코어 16개는 이후 우선순위에 따라 상세화한다.

### 유입 기능

- 랜딩페이지 또는 간이 진단
- SNS 카드뉴스
- 챌린지 미리보기 콘텐츠
- 진단형 콘텐츠
- 청년 정책 체크 콘텐츠

## MVP에서 제외

- 인스타툰 정기 제작
- AI 캐릭터 숏폼 영상
- 앱 내 SNS 공유 자동화
- 친구 초대 리워드
- 계좌 연동
- 금융상품 추천과 가입 대행
- 복잡한 정책금융 신청 자동화
- 포트폴리오 분석과 자산 분산 고도화
- ETF 구성 종목/수수료 검색
- 유료 구독 기능
- 자산 컨설턴트 중개 서비스
- Advanced 자산 분석 구독
- 무작위 보상형 리텐션 이벤트
- 포인트 상점, 랭킹·리더보드, 캐릭터 성장, 일일 출석 보상

## 이유

초기 MVP는 사용자가 첫 금융 행동을 완료하고 다음 주에도 돌아오는지 검증해야 한다. 따라서 재미있는 콘텐츠 제작이나 바이럴 기능보다, 챌린지 화면의 명확성, 완료 기준, 효과 계산, 회고, 재방문 장치가 우선이다.
