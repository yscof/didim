# 07. Roadmap

## 5월

- 문제 정의
- 타겟 페르소나 설정
- 랜딩페이지 제작
- 사회초년생 인터뷰
- 핵심 가설 검증

## 6월

- 유사 서비스 분석
- 금융 도메인 자료 수집
- 사용자 행동 시나리오 분석
- MVP 기능 범위 확정

## 7월

- 서비스 플로우 설계
- UX 와이어프레임 작성
- DB/API 구조 설계
- RAG/LLM 처리 흐름 설계
- 미션 생성 로직 설계

## 8월

- 모바일 앱 핵심 화면 개발
- 백엔드 API 개발
- 자산 단계 진단 구현
- 주간 미션 생성 구현
- AI 코칭 대화 구현

## 9월

- MVP 배포
- 내부 테스트
- 베타 사용자 모집
- 사용자 피드백 수집
- 온보딩, 미션 수락률, 미션 완료율 측정

## 10월

- 피드백 기반 기능 개선
- AI 답변 품질 개선
- 챌린지 라이브러리 확장
- 서비스 안정화

## 11월

- 최종 산출물 정리
- 사업화 가능성 검토
- 제휴 및 유료 기능 검증
- 발표/제안 자료 고도화
